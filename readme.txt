Connect With Telegram .

== Description ==
Connect telegram with your site,user can search and view your last posts / products from telegram.

if you have installed woocommerce so the plugin will show last products to users,too.

This plugin works both with telegram API(SSL Needed) And botsaz.com API.



This plugin is licensed under the [Apache License, Version 2.0](http://www.apache.org/licenses/LICENSE-2.0).

	
== Installation ==

*  From WordPress Dashboard go to "Plugins>> Add New>> Uploads", select 'plugin-zip-file.zip' file and upload it. Or,
*  Extract the zip file, copy and paste to the 'wp-conten/plugins/' directory of your WordPress Installation. 
*  Activate the plugin.
